## [WebRTC](https://www.webrtc-experiment.com/) and ASP.NET MVC!

1. A simple WebRTC one-to-one demo written in ASP.NET MVC (C#) in **September, 2012!**
2. Supports public rooms as well as **password-protected private rooms**!
4. MS-SQL database is used as signaling gateway!

Main Repository: https://github.com/muaz-khan/WebRTC-Experiment

=
